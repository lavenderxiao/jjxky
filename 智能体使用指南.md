# 考研智能体使用指南

## 📋 概述

本项目集成了基于 Gemini AI 的考研智能体系统，提供智能题目生成、答案评估、相似题推荐和AI对话等功能。

## 🚀 快速开始

### 1. 配置API密钥

复制环境变量示例文件：

```bash
cp .env.example .env
```

编辑 `.env` 文件，填入你的 Gemini API Key：

```env
VITE_GEMINI_API_KEY=your_actual_api_key_here
```

### 2. 安装依赖

```bash
npm install
```

### 3. 启动项目

```bash
npm run dev
```

## 🎯 智能体功能

### 1. 知识问答

智能体可以回答经济学考研相关问题，提供专业的学习建议。

**使用场景**：
- AI导师模块的对话功能
- 院校报考咨询
- 学习方法指导

**示例**：
```typescript
import { chatWithTutor } from './services/geminiService';

const response = await chatWithTutor(
  conversationHistory,
  "请解释一下边际效用递减规律",
  userProfile
);
```

### 2. 题目生成

根据科目和知识点智能生成练习题。

**支持题型**：
- 单选题（choice）
- 问答题（qa）
- 计算题（calculation）

**示例**：
```typescript
import { generateQuestion } from './services/geminiService';

const question = await generateQuestion(
  "微观经济学",
  "消费者行为理论",
  "choice"
);
```

### 3. 答案评估

智能评估用户答案，提供详细解析和改进建议。

**评估维度**：
- 正确性判断
- 错误原因分析
- 知识点回顾
- 课程推荐（答错时）

**示例**：
```typescript
import { evaluateAnswer } from './services/geminiService';

const evaluation = await evaluateAnswer(question, userAnswer);
console.log(evaluation.isCorrect); // true/false
console.log(evaluation.detailedExplanation); // 详细解析
```

### 4. 相似题生成

基于原题生成相似题目，用于巩固知识点。

**示例**：
```typescript
import { generateSimilarQuestions } from './services/geminiService';

const similarQuestions = await generateSimilarQuestions(
  originalQuestion,
  5 // 生成5道相似题
);
```

## ⚙️ 配置说明

### 智能体配置

配置文件位于 `src/config/agent.config.ts`：

```typescript
export const AGENT_CONFIG = {
  // 基础配置
  name: "经济学考研AI导师",
  model: "gemini-1.5-pro",
  temperature: 0.7,

  // 功能开关
  features: {
    knowledgeQA: true,
    questionGeneration: true,
    answerEvaluation: true,
    similarQuestions: true,
    // ...
  },

  // 限制配置
  limits: {
    maxQuestionsPerSession: 50,
    questionGenerationTimeout: 10000,
    // ...
  }
};
```

### Prompt模板

Prompt模板位于 `src/prompts/templates.ts`，可以根据需要自定义：

- `SYSTEM_PROMPT` - 系统角色定位
- `CHAT_PROMPT_TEMPLATE` - 对话模板
- `QUESTION_GENERATION_TEMPLATE` - 题目生成模板
- `ANSWER_EVALUATION_TEMPLATE` - 答案评估模板
- `SIMILAR_QUESTIONS_TEMPLATE` - 相似题生成模板

## 🛡️ 错误处理

智能体内置了完善的错误处理和降级机制：

### 错误类型

- `NETWORK_ERROR` - 网络连接失败
- `TIMEOUT_ERROR` - 请求超时
- `PARSE_ERROR` - 响应解析失败
- `VALIDATION_ERROR` - 数据验证失败
- `RATE_LIMIT_ERROR` - 请求频率限制
- `UNKNOWN_ERROR` - 未知错误

### 降级策略

当AI服务不可用时，系统会自动切换到降级模式：

1. **题目生成降级**：使用预设的模拟题库
2. **答案评估降级**：使用简单的规则匹配
3. **对话降级**：使用预设的回复模板

### 重试机制

智能体会自动重试失败的请求：

- 最大重试次数：3次
- 重试延迟：1秒（指数退避）
- 超时时间：可配置

## 📊 性能优化

### 1. 缓存机制

智能体内置了题目缓存，减少重复生成：

```typescript
// 清空缓存
import { clearAgentCache } from './services/geminiService';
clearAgentCache();
```

**缓存配置**：
- 最大缓存条目：100
- 每个知识点缓存题目数：10
- 缓存过期时间：3600秒

### 2. 超时控制

不同操作有不同的超时时间：

- 题目生成：10秒
- 答案评估：8秒
- AI对话：15秒

### 3. 质量控制

题目生成后会进行质量验证：

- 必填字段检查
- 内容长度检查
- 选项合理性检查
- 解析完整性检查

不合格的题目会自动重新生成。

## 🔧 开发指南

### 扩展智能体功能

1. 在 `src/services/aiAgent.ts` 中添加新方法
2. 在 `src/prompts/templates.ts` 中添加对应的Prompt模板

```typescript

  const question = await generateQuestion(subject, kp, type);

  if (error instanceof AgentError) {
    // 根据错误类型处理
    switch (error.type) {
      case AgentErrorType.TIMEOUT_ERROR:
        // 提示用户重试
        break;
      case AgentErrorType.RATE_LIMIT_ERROR:
        // 提示用户稍后再试
        break;
      default:
        // 使用降级方案
    }
  }
}
```

### 3. 性能优化

- 合理使用缓存
- 避免频繁调用AI接口
- 使用批量操作
- 实现请求去重

## 🐛 故障排查

### 问题1：API调用失败

**症状**：所有AI功能都不可用

**解决方案**：
1. 检查 `.env` 文件中的API密钥是否正确
2. 检查网络连接
3. 查看浏览器控制台的错误信息
4. 确认Gemini API配额是否用尽

### 问题2：题目质量不佳

**症状**：生成的题目不符合要求

**解决方案**：
1. 调整 `temperature` 参数（降低以提高稳定性）
2. 优化Prompt模板，提供更多示例
3. 增加质量验证规则
4. 使用更强大的模型（如gemini-1.5-pro）

### 问题3：响应速度慢

**症状**：AI响应时间过长

**解决方案**：
1. 启用缓存机制
2. 使用更快的模型（如gemini-1.5-flash）
3. 减少Prompt长度
4. 调整超时时间
