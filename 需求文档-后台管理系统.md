# 后台管理系统需求文档（PRD）
## 经济学考研AI助手 - 管理后台

---

## 1. 系统概述

### 1.1 项目背景
为了支持经济学考研AI助手前端系统的内容管理和配置，需要开发一套完整的后台管理系统。管理员可以通过后台系统配置智能体信息、管理院校数据、维护题库、设置科目知识点等，实现内容的灵活配置和动态更新。

### 1.2 系统目标
- 提供可视化的内容管理界面，降低运营成本
- 支持智能体配置、院校数据、题库、知识点的全生命周期管理
- 实现数据的批量导入导出，提高数据维护效率
- 提供数据统计和分析功能，支持运营决策
- 确保数据安全和权限管理

### 1.3 用户角色
- **超级管理员**：拥有所有权限，管理系统配置和用户
- **内容管理员**：管理题库、知识点、教材等内容
- **数据管理员**：管理院校、分数线、招生数据
- **运营人员**：查看统计数据，导出报表

## 3. 详细功能需求

### 3.1 系统配置模块

#### 3.1.1 智能体配置

**功能描述**
管理前端系统的智能体基本信息，包括名称、图标、欢迎语等。

**字段设计**
```typescript
{
  id: string,
  name: string,              // 智能体名称，如"经济学考研AI助手"
  icon: string,              // 图标URL或图标代码
  iconType: 'url' | 'emoji' | 'lucide', // 图标类型
  welcomeMessage: string,    // 欢迎语
  description: string,       // 系统描述
  primaryColor: string,      // 主题色（HEX）
  status: 'active' | 'inactive', // 状态
  createdAt: Date,
  updatedAt: Date
}
```

**功能清单**
- [ ] 智能体名称配置（实时预览）
- [ ] 图标上传/选择（支持图片上传、Emoji、Lucide图标库）
- [ ] 欢迎语配置（支持多行文本）
- [ ] 主题色配置（颜色选择器）
- [ ] 系统描述配置
- [ ] 配置历史版本管理
- [ ] 配置发布/回滚功能

**界面设计要点**
- 左侧配置表单，右侧实时预览效果
- 图标选择支持三种方式切换（上传/Emoji/图标库）
- 颜色选择器支持预设色板和自定义输入

#### 3.1.2 系统参数设置

**功能描述**
配置系统全局参数，如AI模型参数、题目生成规则等。

**配置项**
- AI模型选择（Gemini Pro / Gemini Flash）
- 题目生成数量限制（每次练习最多题数）
- 答案评估严格度（宽松/标准/严格）
- 相似题生成数量（默认5题）
- 缓存策略配置
- 错题本容量限制

#### 3.1.3 API配置管理

**功能描述**
管理第三方API密钥和配置。

**配置项**
- Gemini API Key管理
- API调用频率限制
- API超时时间设置
- 降级策略配置

## 7. 非功能需求

### 7.1 性能要求
- 页面加载时间 < 2秒
- API响应时间 < 500ms
- 表格渲染支持虚拟滚动（10000+数据）
- 文件上传支持断点续传
- 批量导入支持异步处理

### 7.2 安全要求
- JWT认证 + Refresh Token
- 密码加密存储（bcrypt）
- 接口权限校验
- SQL注入防护
- XSS攻击防护
- CSRF防护
- 敏感操作二次确认
- 操作日志记录

### 7.3 可用性要求
- 界面友好，操作简单
- 错误提示清晰
- 支持键盘快捷键
- 支持批量操作
- 支持数据导入导出
- 支持数据备份恢复

### 7.4 兼容性要求
- 浏览器：Chrome、Edge、Firefox、Safari最新版本
- 分辨率：1920x1080、1366x768
