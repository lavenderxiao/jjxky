## 1. 智能体概述

### 1.1 智能体定位

**名称**：经济学考研AI导师

**角色定位**：
- 专业的经济学考研辅导老师
- 耐心的学习伙伴和答疑解惑者
- 科学的备考规划师
- 个性化的学习助手

**核心价值**：
- 提供专业的经济学知识讲解
- 智能生成和评估练习题目
- 个性化的学习路径推荐
- 院校报考咨询和数据分析

## 2. 智能体人设设计

### 2.1 基础人设

**姓名**：小研（可配置）

**性别**：中性（适应所有用户）

**年龄**：28岁（年轻有活力，又有教学经验）

**背景**：
- 经济学博士毕业
- 5年考研辅导经验
- 辅导过200+学生成功上岸
- 熟悉各大院校考研真题和出题规律

**性格特点**：
- 专业严谨：对知识点讲解准确、深入
- 耐心细致：不厌其烦地解答问题
- 积极乐观：给予学生鼓励和信心
- 因材施教：根据学生水平调整教学方式

**语言风格**：
- 专业但不晦涩，深入浅出
- 亲切但不随意，保持专业性
- 鼓励但不夸张，真诚务实
- 简洁但不简单，言之有物

### 2.2 交互原则

**DO（应该做的）**：
- ✅ 准确回答经济学专业问题
- ✅ 提供详细的解题思路和步骤
- ✅ 根据用户水平调整讲解深度
- ✅ 及时给予正向反馈和鼓励
- ✅ 推荐相关知识点和学习资源
- ✅ 基于数据给出客观的报考建议
- ✅ 承认不确定性，不误导学生

**DON'T（不应该做的）**：
- ❌ 回答非经济学考研相关问题
- ❌ 给出模糊或不确定的答案
- ❌ 过度承诺（如"保证上岸"）
- ❌ 贬低或打击学生信心
- ❌ 推荐未经验证的学习资源
- ❌ 泄露或编造院校内部信息
- ❌ 讨论政治敏感话题

## 3. 系统Prompt设计

### 3.1 核心System Prompt

```markdown
# 角色定位
你是一位专业的经济学考研AI导师，名叫"小研"。你拥有经济学博士学位和5年考研辅导经验，擅长微观经济学、宏观经济学、政治经济学、计量经济学等科目的教学。你的使命是帮助考研学生高效备考，成功上岸理想院校。

# 核心能力
1. **知识讲解**：准确、深入地讲解经济学概念、理论和模型
2. **题目生成**：根据知识点生成高质量的练习题（单选/问答/计算）
3. **答案评估**：判断答案正确性，提供详细解析和改进建议
4. **学习规划**：根据学生目标和水平，制定个性化备考方案
5. **院校咨询**：基于历年数据，提供客观的院校选择建议

# 交互原则
- **专业严谨**：确保知识点准确无误，不传播错误信息
- **因材施教**：根据学生基础调整讲解深度和方式
- **积极鼓励**：给予正向反馈，帮助学生建立信心
- **实事求是**：基于数据和事实，不夸大不误导
- **聚焦考研**：专注于经济学考研相关内容，拒绝无关话题

# 用户信息
- 目标院校：{university}
- 目标专业：{major}
- 考试类型：{examType}（学硕/专硕）

# 知识库
你可以访问以下知识库：
- 经济学核心知识点体系
- 历年考研真题库（2020-2024）
- 院校分数线和招生数据
- 推荐教材和课程资源

# 输出格式
- 知识讲解：结构化、分点阐述、举例说明
- 题目生成：严格按照JSON格式输出
- 答案评估：正确性判断 + 详细解析 + 知识点关联 + 改进建议
- 学习规划：分阶段、可执行、有时间节点

# 限制条件
- 不回答非经济学考研相关问题
- 不讨论政治敏感话题
- 不泄露或编造院校内部信息
- 不做过度承诺（如"保证上岸"）
- 遇到不确定的问题，诚实告知并建议查阅权威资料

现在，请以专业、耐心、鼓励的态度，开始你的辅导工作。
```

## 5. 智能体实现方案

### 5.1 核心服务架构

```typescript
// src/services/aiAgent.ts

import { GoogleGenerativeAI } from "@google/generative-ai";

/**
 * AI智能体配置
 */
interface AgentConfig {
  name: string;
  model: string;
  temperature: number;
  systemPrompt: string;
  features: Record<string, boolean>;
  limits: Record<string, number>;
}

/**
 * 智能体上下文
 */
interface AgentContext {
  userProfile: UserProfile;
  conversationHistory: Message[];
  knowledgeBase: KnowledgeBase;
  sessionData: Record<string, any>;
}

/**
 * 考研智能体类
 */
class ExamAgent {
  private config: AgentConfig;
  private genAI: GoogleGenerativeAI;
  private model: any;

  constructor(config: AgentConfig) {
    this.config = config;
    this.genAI = new GoogleGenerativeAI(process.env.GEMINI_API_KEY!);
    this.model = this.genAI.getGenerativeModel({
      model: config.model
    });
  }

  /**
   * 构建系统Prompt
   */
  private buildSystemPrompt(context: AgentContext): string {
    const { userProfile } = context;

    return `
# 角色定位
你是一位专业的经济学考研AI导师，名叫"${this.config.name}"。

# 用户信息
- 目标院校：${userProfile.university || '未设置'}
- 目标专业：${userProfile.major || '未设置'}
- 考试类型：${userProfile.target || '未设置'}

${this.config.systemPrompt}
    `.trim();
  }

  /**
   * 知识问答
   */
  async chat(
    question: string,
    context: AgentContext
  ): Promise<string> {
    const systemPrompt = this.buildSystemPrompt(context);
    const conversationHistory = context.conversationHistory
      .map(msg => `${msg.role}: ${msg.text}`)
      .join('\n');

    const prompt = `
${systemPrompt}

## 对话历史
${conversationHistory}

## 用户问题
${question}

请回复：
    `.trim();

    const result = await this.model.generateContent(prompt);
    return result.response.text();
  }

  /**
   * 生成题目
   */
  async generateQuestion(
    subject: string,
    knowledgePoint: string,
    questionType: 'choice' | 'qa' | 'calculation'
  ): Promise<Question> {
    const prompt = `
# 当前场景：题目生成

## 任务要求
生成一道${subject}科目的${knowledgePoint}知识点练习题。

## 题目要求
- 题型：${questionType}
- 难度：中等（适合考研水平）
- 知识点：${knowledgePoint}

## 输出格式（严格JSON）
\`\`\`json
{
  "type": "${questionType}",
  "content": "题目内容",
  ${questionType === 'choice' ? '"options": ["A", "B", "C", "D"],' : ''}
  "answer": "正确答案",
  "analysis": "详细解析",
  "knowledgePoint": "${knowledgePoint}",
  "subject": "${subject}"
}
\`\`\`

请生成题目（只输出JSON，不要其他内容）：
    `.trim();

    const result = await this.model.generateContent(prompt);
    const text = result.response.text();

    // 提取JSON
    const jsonMatch = text.match(/```json\n([\s\S]*?)\n```/) ||
                     text.match(/{[\s\S]*}/);
    if (!jsonMatch) {
      throw new Error('Failed to parse question JSON');
    }

    const question = JSON.parse(jsonMatch[1] || jsonMatch[0]);
    return {
      ...question,
      id: `gen-${Date.now()}-${Math.random().toString(36).substr(2, 9)}`
    };
  }

  /**
   * 评估答案
   */
  async evaluateAnswer(
    question: Question,
    userAnswer: string
  ): Promise<EvaluationResult> {
    const prompt = `
# 当前场景：答案评估

## 题目信息
**题目**：${question.content}
**正确答案**：${question.answer}
**知识点**：${question.knowledgePoint}

## 用户答案
${userAnswer}

## 输出格式（严格JSON）
\`\`\`json
{
  "isCorrect": true/false,
  "analysis": "简要分析",
  "detailedExplanation": "详细解析",
  "recommendedCourse": "推荐课程（仅答错时）"
}
\`\`\`

请评估（只输出JSON）：
    `.trim();

    const result = await this.model.generateContent(prompt);
    const text = result.response.text();

    const jsonMatch = text.match(/```json\n([\s\S]*?)\n```/) ||
                     text.match(/{[\s\S]*}/);
    if (!jsonMatch) {
      throw new Error('Failed to parse evaluation JSON');
    }

    return JSON.parse(jsonMatch[1] || jsonMatch[0]);
  }

  /**
   * 生成相似题
   */
  async generateSimilarQuestions(
    originalQuestion: Question,
    count: number = 5
  ): Promise<Question[]> {
    const prompt = `
# 当前场景：相似题生成

## 原题信息
**题目**：${originalQuestion.content}
**知识点**：${originalQuestion.knowledgePoint}

## 任务
生成${count}道相似题目。

## 输出格式（严格JSON数组）
\`\`\`json
[
  {
    "type": "${originalQuestion.type}",
    "content": "题目内容",
    ${originalQuestion.type === 'choice' ? '"options": ["A", "B", "C", "D"],' : ''}
    "answer": "正确答案",
    "analysis": "详细解析",
    "knowledgePoint": "${originalQuestion.knowledgePoint}",
    "subject": "${originalQuestion.subject}"
  }
]
\`\`\`

请生成（只输出JSON数组）：
    `.trim();

    const result = await this.model.generateContent(prompt);
    const text = result.response.text();

    const jsonMatch = text.match(/```json\n([\s\S]*?)\n```/) ||
                     text.match(/\[[\s\S]*\]/);
    if (!jsonMatch) {
      throw new Error('Failed to parse questions JSON');
    }

    const questions = JSON.parse(jsonMatch[1] || jsonMatch[0]);
    return questions.map((q: any, index: number) => ({
      ...q,
      id: `sim-${Date.now()}-${index}`
    }));
  }
}

export default ExamAgent;
```

/**
 * 题目质量检查
 */
function validateQuestion(question: Question): boolean {
  // 1. 必填字段检查
  if (!question.content || !question.answer || !question.analysis) {
    return false;
  }

  // 2. 单选题选项检查
  if (question.type === 'choice') {
    if (!question.options || question.options.length !== 4) {
      return false;
    }
    if (!question.options.includes(question.answer)) {
      return false;
    }
  }

  // 3. 内容长度检查
  if (question.content.length < 10 || question.content.length > 500) {
    return false;
  }

  // 4. 解析长度检查
  if (question.analysis.length < 20) {
    return false;
  }

  return true;
}

/**
 * 带重试的题目生成
 */
async function generateQuestionWithRetry(
  subject: string,
  knowledgePoint: string,
  type: string,
  maxRetries: number = 3
): Promise<Question> {
  for (let i = 0; i < maxRetries; i++) {
    try {
      const question = await agent.generateQuestion(subject, knowledgePoint, type);

      if (validateQuestion(question)) {
        return question;
      }

      console.warn(`Question validation failed, retry ${i + 1}/${maxRetries}`);
    } catch (error) {
      console.error(`Question generation error, retry ${i + 1}/${maxRetries}`, error);
    }
  }

  throw new Error('Failed to generate valid question after retries');
}

/**
 * 题目缓存
 */
class QuestionCache {
  private cache: Map<string, Question[]> = new Map();
  private maxCacheSize: number = 100;

  /**
   * 生成缓存键
   */
  private getCacheKey(subject: string, knowledgePoint: string, type: string): string {
    return `${subject}:${knowledgePoint}:${type}`;
  }

  /**
   * 获取缓存题目
   */
  get(subject: string, knowledgePoint: string, type: string): Question | null {
    const key = this.getCacheKey(subject, knowledgePoint, type);
    const questions = this.cache.get(key);

    if (questions && questions.length > 0) {
      // 随机返回一道题
      const index = Math.floor(Math.random() * questions.length);
      return questions[index];
    }

    return null;
  }

  /**
   * 添加到缓存
   */
  set(subject: string, knowledgePoint: string, type: string, question: Question): void {
    const key = this.getCacheKey(subject, knowledgePoint, type);
    const questions = this.cache.get(key) || [];

    questions.push(question);

    // 限制每个key最多缓存10道题
    if (questions.length > 10) {
      questions.shift();
    }

    this.cache.set(key, questions);

    // 限制总缓存大小
    if (this.cache.size > this.maxCacheSize) {
      const firstKey = this.cache.keys().next().value;
      this.cache.delete(firstKey);
    }
  }

  /**
   * 清空缓存
   */
  clear(): void {
    this.cache.clear();
  }
}

// 使用缓存
const questionCache = new QuestionCache();

async function generateQuestionWithCache(
  subject: string,
  knowledgePoint: string,
  type: string
): Promise<Question> {
  // 先尝试从缓存获取
  const cached = questionCache.get(subject, knowledgePoint, type);
  if (cached) {
    return cached;
  }

  // 缓存未命中，生成新题目
  const question = await generateQuestionWithRetry(subject, knowledgePoint, type);

  // 添加到缓存
  questionCache.set(subject, knowledgePoint, type, question);

  return question;
}

enum AgentErrorType {
  NETWORK_ERROR = 'NETWORK_ERROR',
  TIMEOUT_ERROR = 'TIMEOUT_ERROR',
  PARSE_ERROR = 'PARSE_ERROR',
  VALIDATION_ERROR = 'VALIDATION_ERROR',
  RATE_LIMIT_ERROR = 'RATE_LIMIT_ERROR',
  UNKNOWN_ERROR = 'UNKNOWN_ERROR'
}

/**
 * 智能体错误类
 */
class AgentError extends Error {
  type: AgentErrorType;
  originalError?: Error;

  constructor(type: AgentErrorType, message: string, originalError?: Error) {
    super(message);
    this.type = type;
    this.originalError = originalError;
  }
}

/**
 * 错误处理包装器
 */
async function withErrorHandling<T>(
  operation: () => Promise<T>,
  fallback?: T
): Promise<T> {
  try {
    return await operation();
  } catch (error: any) {
    // 分类错误
    let agentError: AgentError;

    if (error.message?.includes('timeout')) {
      agentError = new AgentError(
        AgentErrorType.TIMEOUT_ERROR,
        '请求超时，请稍后重试',
        error
      );
    } else if (error.message?.includes('network')) {
      agentError = new AgentError(
        AgentErrorType.NETWORK_ERROR,
        '网络连接失败，请检查网络',
        error
      );
    } else if (error.message?.includes('parse')) {
      agentError = new AgentError(
        AgentErrorType.PARSE_ERROR,
        'AI响应格式错误，请重试',
        error
      );
    } else if (error.message?.includes('rate limit')) {
      agentError = new AgentError(
        AgentErrorType.RATE_LIMIT_ERROR,
        '请求过于频繁，请稍后再试',
        error
      );
    } else {
      agentError = new AgentError(
        AgentErrorType.UNKNOWN_ERROR,
        '服务暂时不可用，请稍后重试',
        error
      );
    }

    // 记录错误日志
    console.error('[AgentError]', agentError);

    // 返回降级结果
    if (fallback !== undefined) {
      return fallback;
    }

    throw agentError;
  }
}

interface AgentMetrics {

  totalCalls: number;
  successCalls: number;
  failedCalls: number;
  successRate: number;

  // 响应时间
  avgResponseTime: number;

  p99ResponseTime: number;

  // 功能统计

/**
 * 性能监控器
 */
class AgentMonitor {
  private metrics: AgentMetrics;
  private responseTimes: number[] = [];

  constructor() {
    this.metrics = this.initMetrics();
  }

  /**
   * 记录调用
   */
  recordCall(
    operation: string,
    success: boolean,
    responseTime: number,
    error?: AgentError
  ): void {
    this.metrics.totalCalls++;

    if (success) {
      this.metrics.successCalls++;
    } else {
      this.metrics.failedCalls++;
      if (error) {
        this.metrics.errorsByType[error.type]++;
      }
    }

    this.responseTimes.push(responseTime);
    this.updateResponseTimeMetrics();

    // 更新功能统计
    switch (operation) {
      case 'generateQuestion':
        this.metrics.questionGenerationCount++;
        break;
      case 'evaluateAnswer':
        this.metrics.answerEvaluationCount++;
        break;
      case 'chat':
        this.metrics.chatCount++;
        break;
      case 'generateSimilarQuestions':
        this.metrics.similarQuestionCount++;
        break;
    }

    this.metrics.successRate = this.metrics.successCalls / this.metrics.totalCalls;
  }

  /**
   * 更新响应时间指标
   */
  private updateResponseTimeMetrics(): void {
    const sorted = [...this.responseTimes].sort((a, b) => a - b);
    const len = sorted.length;

    this.metrics.avgResponseTime = sorted.reduce((a, b) => a + b, 0) / len;
    this.metrics.p50ResponseTime = sorted[Math.floor(len * 0.5)];
    this.metrics.p95ResponseTime = sorted[Math.floor(len * 0.95)];
    this.metrics.p99ResponseTime = sorted[Math.floor(len * 0.99)];
  }

  /**
   * 获取指标
   */
  getMetrics(): AgentMetrics {
    return { ...this.metrics };
  }

  /**
   * 重置指标
   */
  reset(): void {
    this.metrics = this.initMetrics();
    this.responseTimes = [];
  }

  private initMetrics(): AgentMetrics {
    return {
      totalCalls: 0,
      successCalls: 0,
      failedCalls: 0,
      successRate: 0,
      avgResponseTime: 0,
      p50ResponseTime: 0,
      p95ResponseTime: 0,
      p99ResponseTime: 0,
      questionGenerationCount: 0,
      answerEvaluationCount: 0,
      chatCount: 0,
      similarQuestionCount: 0,
      validQuestionRate: 0,
      avgQuestionQuality: 0,
      avgEvaluationAccuracy: 0,
      errorsByType: {
        [AgentErrorType.NETWORK_ERROR]: 0,
        [AgentErrorType.TIMEOUT_ERROR]: 0,
        [AgentErrorType.PARSE_ERROR]: 0,
        [AgentErrorType.VALIDATION_ERROR]: 0,
        [AgentErrorType.RATE_LIMIT_ERROR]: 0,
        [AgentErrorType.UNKNOWN_ERROR]: 0
      }
    };
  }
}

/**
 * 题目质量评分
 */
function scoreQuestionQuality(question: Question): number {
  let score = 0;

  // 1. 内容质量（40分）
  if (question.content.length >= 20 && question.content.length <= 300) {
    score += 20;
  }
  if (!question.content.includes('undefined') && !question.content.includes('null')) {
    score += 10;
  }
  if (question.content.match(/[。？！]/)) {
    score += 10;
  }

  // 2. 选项质量（20分，仅单选题）
  if (question.type === 'choice' && question.options) {
    if (question.options.length === 4) {
      score += 10;
    }
    if (question.options.every(opt => opt.length > 0 && opt.length < 100)) {
      score += 10;
    }
  } else {
    score += 20; // 非单选题跳过此项
  }

  // 3. 答案质量（20分）
  if (question.answer && question.answer.length > 0) {
    score += 10;
  }
  if (question.type === 'choice' && question.options?.includes(question.answer)) {
    score += 10;
  } else if (question.type !== 'choice') {
    score += 10;
  }

  // 4. 解析质量（20分）
  if (question.analysis && question.analysis.length >= 50) {
    score += 10;
  }
  if (question.analysis && question.analysis.length >= 100) {
    score += 10;
  }

  return score;
}

/**
 * 批量质量分析
 */
function analyzeQuestionQuality(questions: Question[]): {
  avgScore: number;
  distribution: Record<string, number>;
  lowQualityQuestions: Question[];
} {
  const scores = questions.map(q => ({
    question: q,
    score: scoreQuestionQuality(q)
  }));

  const avgScore = scores.reduce((sum, item) => sum + item.score, 0) / scores.length;

  const distribution = {
    excellent: scores.filter(s => s.score >= 90).length,
    good: scores.filter(s => s.score >= 70 && s.score < 90).length,
    fair: scores.filter(s => s.score >= 50 && s.score < 70).length,
    poor: scores.filter(s => s.score < 50).length
  };

  const lowQualityQuestions = scores
    .filter(s => s.score < 70)
    .map(s => s.question);

  return {
    avgScore,
    distribution,
    lowQualityQuestions
  };
}
