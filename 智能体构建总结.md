## 🎯 智能体核心功能

### 1. 知识问答
```typescript
const response = await chatWithTutor(
  conversationHistory,
  "请解释边际效用递减规律",
  userProfile
);
```

**特性**：
- 基于用户画像的个性化回复
- 上下文理解（多轮对话）
- 院校数据分析
- 专业、耐心、鼓励的语气

### 2. 题目生成
```typescript
const question = await generateQuestion(
  "微观经济学",
  "消费者行为理论",
  "choice"
);
```

**特性**：
- 支持3种题型（单选/问答/计算）
- 自动质量验证
- 失败自动重试（最多3次）
- 题目缓存机制
- 符合考研真题风格

### 3. 答案评估
```typescript
const evaluation = await evaluateAnswer(question, userAnswer);
```

**特性**：
- 智能判断正确性
- 详细解析和错误分析
- 知识点回顾
- 课程推荐（答错时）

### 4. 相似题生成
```typescript
const similarQuestions = await generateSimilarQuestions(
  originalQuestion,
  5
);
```

**特性**：
- 基于原题生成相似题
- 保持知识点和难度一致
- 题目形式可变化
- 批量生成（最多10道）

## 🚀 快速开始

### 1. 配置API密钥

```bash
# 复制环境变量示例
cp .env.example .env

# 编辑 .env 文件，填入API密钥
VITE_GEMINI_API_KEY=your_api_key_here
```

### 2. 安装依赖

```bash
npm install
```

### 3. 启动项目

```bash
npm run dev
```

### 4. 测试智能体（可选）

```bash
# 运行测试脚本
npx tsx test-agent.ts
```
