# 经济学考研智能体系统

## 系统架构

```
智能体系统
├── 核心层 (aiAgent.ts)
│   ├── ExamAgent 类
│   ├── 题目生成引擎
│   ├── 答案评估引擎
│   └── 对话管理引擎
├── 服务层 (geminiService.ts)
│   ├── API 集成
│   ├── 降级处理
│   └── 缓存管理
├── 配置层 (agent.config.ts)
│   └── 功能开关/限制/质量控制
└── Prompt层 (templates.ts)
    └── 场景化提示词模板
```

## 核心功能

### 1. 智能题目生成
- 支持单选题、问答题、计算题
- 基于知识点精准生成
- 自动质量验证
- 智能缓存机制

### 2. 答案智能评估
- 准确性判断
- 详细解析生成
- 课程推荐
- 学习建议

### 3. 相似题目生成
- 基于原题生成变体
- 保持知识点一致
- 难度可控

### 4. AI导师对话
- 知识问答
- 院校咨询
- 学习规划
- 志愿填报建议

## 技术特性

### 缓存机制
- 题目缓存：每个知识点缓存10道题
- LRU淘汰策略
- 最大缓存100个条目

### 容错机制
- 3次自动重试
- 指数退避策略
- 降级模式（Fallback）
- 超时保护

### 质量控制
- 题目长度验证
- 选项合理性检查
- 答案准确性验证
- 解析完整性检查
