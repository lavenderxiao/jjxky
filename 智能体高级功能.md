# 智能体高级使用指南

## 快速开始

### 1. 环境配置

在 `.env` 文件中配置 Gemini API Key：

```bash
VITE_GEMINI_API_KEY=your_api_key_here
```

### 2. 基础使用

```typescript
import { generateQuestion, evaluateAnswer, chatWithTutor } from '@/services/geminiService'

// 生成题目
const question = await generateQuestion('微观经济学', '需求理论', 'choice')

// 评估答案
const result = await evaluateAnswer(question, userAnswer)

// AI对话
const response = await chatWithTutor(history, message, userProfile)
```

## 高级功能

### 学习规划

```typescript
import ExamAgent from '@/services/aiAgent'
import { StudyPlanner } from '@/services/studyPlanner'

const agent = new ExamAgent(apiKey)
const planner = new StudyPlanner(agent)

const plan = await planner.generateStudyPlan({
  userProfile,
  currentDate: new Date(),
  examDate: new Date('2024-12-23'),
  weeklyHours: 40,
  strengths: ['微观经济学'],
  weaknesses: ['计量经济学']
})
```

### 院校咨询

```typescript
import { UniversityConsultant } from '@/services/universityConsultant'

const consultant = new UniversityConsultant(agent)

// 分析目标院校
const analysis = await consultant.analyzeUniversity({
  userProfile,
  enrollmentData,
  scoreData,
  userScore: {
    politics: 70,
    english: 75,
    math: 120,
    professional: 130
  }
})

// 推荐备选院校
const alternatives = await consultant.recommendAlternatives({
  userProfile,
  enrollmentData,
  scoreData,
  count: 5
})
```

### 性能监控

```typescript
import { agentMonitor } from '@/services/agentMonitor'

// 获取性能指标
const metrics = agentMonitor.getMetrics()
console.log('成功率:', metrics.successfulRequests / metrics.totalRequests)
console.log('平均响应时间:', metrics.averageResponseTime, 'ms')
console.log('缓存命中率:', metrics.cacheHitRate, '%')
```
